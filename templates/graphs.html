Regresión Lineal

Modela la relación entre una variable dependiente Y y una independiente X ajustando una línea recta.

y = mx + b   →   y = 2.5x + 5.0
Datos reales Línea de regresión (predicción)

¿Cómo funciona?

  • Encuentra la línea recta que mejor se ajusta a los datos
  • Minimiza la suma de errores al cuadrado (Mínimos Cuadrados Ordinarios)
  • Sirve para predecir valores numéricos continuos (precios, temperaturas, etc.)
  • Problema: Si la aplicamos a clasificación, da probabilidades fuera de [0,1]

Regresión Logística

A pesar del nombre, es un clasificador binario. Usa la función sigmoide para convertir cualquier valor en una probabilidad entre 0 y 1.

p(y=1|x) = 1 / (1 + e-(z))   donde   z = w·x + b
Clase 0 (Negativo) Clase 1 (Positivo) Curva Sigmoide

¿Cómo funciona?

  • Calcula una combinación lineal de las features: z = w·x + b
  • Aplica la función sigmoide σ(z) para convertir z en probabilidad
  • Si p > 0.5 → Clase 1, si p ≤ 0.5 → Clase 0
  • Los pesos w se ajustan con gradiente descendente para minimizar el error

¿Por qué Logística y no Lineal para clasificación?

La regresión lineal no funciona para clasificación porque sus predicciones no están acotadas entre 0 y 1, y un outlier puede distorsionar todo el modelo.

❌ Con Lineal

  • Predice valores menores a 0 o mayores a 1
  • Un outlier desplaza toda la línea
  • No puede modelar relaciones no lineales
  • Los errores no se distribuyen normalmente

✅ Con Logística

  • La probabilidad siempre entre 0 y 1
  • Robusta a outliers
  • La sigmoide captura relaciones no lineales
  • Diseñada específicamente para clasificación binaria

La Función Sigmoide en detalle

La función sigmoide σ(z) = 1/(1+e⁻ᶻ) convierte cualquier número real en una probabilidad entre 0 y 1.

σ(z) = 1 / (1 + e-z)     z → -∞  ⇒  σ(z) → 0     z → +∞  ⇒  σ(z) → 1     z = 0  ⇒  σ(z) = 0.5
σ(z) = 1/(1+e⁻ᶻ) Frontera: z = 0, p = 0.5

Propiedades importantes

  • Rango: Siempre devuelve valores entre 0 y 1
  • Simetría: σ(-z) = 1 - σ(z) — simétrica respecto a (0, 0.5)
  • Derivada: σ'(z) = σ(z) · (1 - σ(z)) — fácil de calcular para gradiente descendente
  • Lineal cerca de 0: Aproximadamente lineal entre z = -2 y z = +2
  • Saturación: Para |z| > 5, la curva se aplana (gradiente cercano a 0)
